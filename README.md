# ICF_Crud_REST_API

This project includes the CRUD operations performed on database using REST API.
Spring boot + Hibernate along with Maven tool is used in this project.

Steps to be followed to run the project:

1. Download the source code and export it to Eclipse IDE or convert the code into jar file and run it on terminal
2. Run the DemoApplication.java file and open http://localhost:4000/ in your browser.

List of Endpoints:

1. Home page/ list of all employees => http://localhost:4000/
2. List of all Dependants => http://localhost:4000/dependants
3. List of all Educational Qualifications => http://localhost:4000/qualifications

A responsive page is added in which we can add new employee/dependant/educational_qualification by clicking on buttons which would call further endpoint and display templates.

JUnitTests has been added in DemoApplicationTests.java

Added simple front end and backend validations for the inputs.
